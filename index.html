Maximum G-Load Relational N-Back (0.95+)

This is the most cognitively demanding n-back variant possible. Each 5-word premise encodes multiple interfering relationships that must be mentally simulated simultaneously.

Core Concept: Relational Interference

Each premise describes symbols affecting each other in ways that CREATE INTERFERENCE - solving one relationship changes the constraints on others.

PREMISE STRUCTURE (exactly 5 words): "A-with-B pushes-while-rotating triggers C-and-D" This encodes: • Two bound symbols (A-with-B) • Dual operation (pushes-while-rotating) • Causal relation (triggers) • Target pair (C-and-D) • Result state (implicit)

What Makes a True Match?

A match occurs when premises share the same RELATIONAL TOPOLOGY - the pattern of who-affects-whom and how those effects interfere with each other.

Trial 1: "A-with-B pushes-while-rotating triggers C-and-D" Trial 3: "X-and-Y pulls-while-spinning enables Z-with-W" MATCH! Because: • Same topology: bound-pair affects bound-pair • Same interference: push+rotate ≈ pull+spin • Same cascade: immediate causal trigger • Conservation preserved: energy transfers between pairs

Mental Simulation Required

You cannot pattern-match! You must:

  • Track how operations interfere (pushing-while-rotating creates spiral motion)
  • Compute cascading effects (A affects B affects C)
  • Verify conservation (total energy/momentum preserved)
  • Check temporal consistency (effects propagate in time)

Key Strategy

Build a mental model of the RELATIONAL STRUCTURE, not the specific symbols. Ask yourself:

  • What is the pattern of relationships?
  • How do the operations interfere with each other?
  • What constraints does this create?
  • Does this pattern match the n-back reference?

Difficulty Settings

  • Match Probability: Higher = more matches to practice recognition
  • N-Back Level: How far back to compare (1 = moderate, 3+ = extreme)
  • Response Window: Time to analyze and respond

Remember: This requires maintaining and manipulating multiple interfering transformations in working memory simultaneously - the essence of fluid intelligence.
