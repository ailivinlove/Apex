🧠 Relational Lattice N-Back — Maximum Cognitive Load

This is the most demanding n-back variant ever created. Each 5-word premise encodes multiple symbols in a relational lattice. You must track positions, transformations, and cascading relationships across time.

⚡ How It Works

Each premise describes symbols (A, B, C, etc.) undergoing transformations that affect other symbols. These create chains of relationships that cascade and interfere.

PREMISE STRUCTURE (exactly 5 words): [SYMBOL₁] [OPERATION-AXIS] [RELATION] [SYMBOL₂] [AXIS₂] Example: "A moves-UP affecting B DOWN" ↓ ↓ ↓ ↓ ↓ Symbol Operation Causal Symbol Direction

🎯 Your Task

  1. Listen/Read each premise
  2. Mentally simulate the transformations and relationships
  3. Compare with the premise N trials back
  4. Press SPACE if you detect a relational match
  5. Do nothing if no match exists

🔍 What Makes a MATCH?

A match occurs when the current premise is relationally compatible with the N-back premise:

N-back 1 Example: Trial 1: "A moves-UP affecting B DOWN" Trial 2: "A shifts-DOWN affecting B UP" MATCH! Why? The A→B relationship is preserved with inverted spatial axes.
Multi-Symbol Cascade: Trial 1: "A triggers B which-affects C" Trial 2: "X blocks Y preventing Z" Trial 3: "A enables B which-triggers C" Trial 3 matches Trial 1 (2-back): Similar cascade pattern A→B→C

🧩 Key Concepts

Symbols

  • Letters: A, B, C, X, Y, Z (abstract entities)
  • Colors: RED, BLUE, GREEN (visual entities)
  • Numbers: ONE, TWO, THREE (quantitative entities)

Operations

  • Movement: moves, shifts, rotates, spins
  • Transformation: splits, merges, grows, shrinks
  • Force: pushes, pulls, drives, flows

Relations (How symbols affect each other)

  • affecting: Direct influence
  • triggering: Causes activation
  • blocking: Prevents action
  • enabling: Allows action
  • preventing: Stops entirely

Axes (Spatial/State dimensions)

  • Spatial: UP, DOWN, LEFT, RIGHT, IN, OUT
  • States: HOT, COLD, FAST, SLOW

⚠️ Why This Is So Hard

You cannot pattern-match! The same words create different relationships based on context. You must: • Track each symbol's state across multiple dimensions • Compute cascading effects through relationship chains • Detect latent conflicts that emerge from interactions • Maintain a mental model of the entire relational lattice

🎮 Match Probability Setting

Adjust the "Match probability" slider to control difficulty:

  • 0-20%: Few matches (harder - most are non-matches)
  • 25-35%: Optimal challenge
  • 40-60%: Balanced practice
  • 70-100%: Match-heavy (easier - practice recognizing patterns)

💡 Strategy Tips

  1. Visualize the symbols in 3D space
  2. Track relationships not just positions
  3. Note cascades: If A→B and B→C, then A indirectly affects C
  4. Check inversions: UP/DOWN and LEFT/RIGHT often indicate compatible matches
  5. Detect conflicts: "blocking" contradicts "enabling"
CASCADING EFFECTS: A affects B, B affects C ↓ ↓ A indirectly affects C If later premise breaks B→C link, the entire cascade collapses!

Remember: This tests your working memory's ability to maintain and manipulate complex relational structures. Take your time to mentally simulate before responding!
